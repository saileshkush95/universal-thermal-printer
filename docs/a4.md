# A4 / PDF Printing

Print invoices, receipts, and documents on A4 or Letter paper using PDF output.

## Requirement

Install `pdf-lib` (optional peer dependency):

```sh
npm install pdf-lib
# or
bun add pdf-lib
```

## Usage

Add `format: "a4"` to the `print()` options. The same `PrintSection[]` data works for both thermal and A4 mode.

```ts
import { print } from "universal-thermal-printer";

await print("network", "192.168.1.100", [
  { type: "Init" },
  { type: "Align", value: "center" },
  { type: "Bold", value: true },
  { type: "Size", value: { width: 3, height: 3 } },
  { type: "Text", value: "INVOICE" },
  { type: "Bold", value: false },
  { type: "Size", value: { width: 1, height: 1 } },
  { type: "Line", value: "-" },
  { type: "Text", value: "Item 1 ................ $10.00" },
  { type: "Text", value: "Item 2 ................ $20.00" },
  { type: "Feed", value: 3 },
], { format: "a4", title: "My Store", pageSize: "A4" });
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `format` | `"thermal"` \| `"a4"` | `"thermal"` | Output format |
| `pageSize` | `"A4"` \| `"Letter"` | `"A4"` | Page dimensions |
| `margins` | `number` | `40` | Margins in points (1pt = 1/72 inch) |
| `title` | `string` | — | Centered title at top of page |

## Section Support

| Section | A4 Behavior |
|---------|-------------|
| `Text` | PDF text with auto-word-wrapping |
| `Align` | Left / center / right alignment |
| `Size` | Font size scaled by `max(width, height)` |
| `Bold` | Helvetica-Bold font |
| `Feed` / `FeedDots` | Adds vertical spacing |
| `Line` | Horizontal rule |
| `Init` | Ignored |
| `Cut` / `Drawer` / `Beep` | Ignored (not applicable to A4) |
| `Barcode` / `Qr` | Ignored (not yet supported in PDF) |

## Transports

A4/PDF works with all transports:

- `"network"` — Send PDF bytes to printer over TCP (port 9100)
- `"bluetooth"` — Send PDF bytes over Bluetooth
- `"usb"` — Send PDF bytes over USB
- `"spooler"` — Send PDF bytes to OS print spooler
